Licenças

Tudo o que a Halo Project Brasil cria possui uma licença. Descubra mais sobre elas aqui.

Tema Wordpress

O tema wordpress utilizado possui uma licença GPL v2, e seu código fonte pode ser encontrado no GitHub.

O que isso quer dizer?

Isso quer dizer que você pode baixar, modificar, usar e compartilhar livremente esse código, desde que mantenha a licença GPL v2

Saiba mais sobre a licença GPL versão 2.

Conteúdo

O conteúdo apresentado no website possui uma licença Creative Commons - Atribuição-NãoComercial-CompartilhaIgual 4.0 Internacional.

O que isso quer dizer?

Isso quer dizer que você está livre para compartilhar e adaptar o material, mas que deve dar os devidos créditos à Halo Project Brasil, e indicar se ouve alguma mudança, além de não poder usar esse material para propósitos comerciais, e compartilhar sob a mesma licença (Creative Commons - Atribuição-NãoComercial-CompartilhaIgual 4.0 Internacional).

Saiba mais sobre a licença Creative Commons - Atribuição-NãoComercial-CompartilhaIgual 4.0 Internacional.

Documentação

Assim como o conteúdo do website, a documentação possui uma licença Creative Commons, mas levemente diferente da anterior. Neste caso, é a licença Creative Commons - Atribuição-CompartilhaIgual 4.0 Internacional. Seu código encontra-se no GitHub.

O que isso quer dizer?

Isso quer dizer que você possui as mesmas liberdades descritas anteriormente, mas com um diferencial: está livre para usar para fins comerciais. Essa decisão foi tomada devido ao fato de que é apenas uma documentação, e não representa um conteúdo da mesma forma como no website.

Saiba mais sobre a licença Creative Commons - Atribuição-CompartilhaIgual 4.0 Internacional.

Direitos de Uso da Imagem de Halo

A Microsoft nos permite utilizar a imagem de Halo, desde que seja seguido os seguintes termos:

Halo © Microsoft Corporation. Halo Project Brasil was created under Microsoft's "Game Content Usage Rules" using assets from Halo, and it is not endorsed by or affiliated with Microsoft.

O que isso quer dizer?

Isso quer dizer que nós podemos criar materiais sobre Halo, mas que devemos seguir uma série de regras (link abaixo), e que não recebemos apoio da Microsoft, ou de qualquer outra entidade envolvida.
